## Retail Sales Dashboard

### Objective
Analyze regional sales performance and identify profit-driving categories.

### Tools
- Power BI
- Power Query
- DAX
- Excel

### KPIs
- Total Sales
- Profit Margin
- Top Products
- Regional Performance

### Insights
- South region generated highest profit.
- Technology category had strongest YoY growth.
